# Hello DevOps

Egyszerű Flask alapú "Hello DevOps" alkalmazás, amely HTTP-n elérhető a http://localhost:8080 címen.

## Előkészületek

- Python 3.x
- Git
- (Ajánlott) virtuális környezet

## Telepítés és futtatás

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
